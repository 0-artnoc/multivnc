MultiVNC

A tabbed cross-platform multicast VNC viewer.

by Christian Beier

About

MultiVNC is a cross-platform Multicast-enabled VNC viewer using wxWidgets and libvncclient . It runs on Unix, Mac OS X and Windows. Features include support for most encodings, ZeroConf service discovery and a seamless edge control mode รก la x2vnc.
